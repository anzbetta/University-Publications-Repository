# 🎓 University Publications Management UI

A modern web interface for managing university publications, built with React and Vite.

## 🚀 Overview

This project is a frontend application designed to streamline the process of managing academic publications within a university environment. It provides interfaces for authors, administrators, and public users.

The system allows:
- Authors to submit and manage publications
- Administrators to review and approve content
- Users to browse a public catalog of publications

## 🧩 Features

### 👨‍🎓 Author Dashboard
- Create and submit new publications
- View submission status
- Manage personal publications

### 🛠 Admin Panel
- Review submitted publications
- Approve or reject entries
- Manage publication workflow

### 🌐 Public Catalog
- Browse all approved publications
- View detailed publication information

### 🔐 Authentication UI
- Login interface for users
- Role-based UI rendering
